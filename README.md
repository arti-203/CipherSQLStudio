# CipherSQLStudio

CipherSQLStudio is a browser-based SQL learning platform where users can practice SQL queries against pre-configured assignments and receive intelligent hints.

---

##  Features

- Assignment listing with difficulty and description
- SQL editor using Monaco Editor
- Execute SQL queries against a sandbox PostgreSQL database
- View query results in real-time
- LLM-powered hint system (mocked for security)
- Mobile-first responsive UI using SCSS

---

##  Tech Stack

### Frontend
- React.js (Create React App)
- SCSS (vanilla, mobile-first)
- Monaco Editor

### Backend
- Node.js
- Express.js
- PostgreSQL (sandbox execution)
